# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
リモートリポジトリ＝オンラインの保管庫  
ローカルリポジトリ＝自分のPC内の作業場

## プッシュとマージの違いは何でしょうか？
push＝「リモートに送る」  
merge＝「別の変更を取り込んで1つにまとめる」

## コミットとプッシュの違い
commit＝ローカルで変更を“記録”する  
push＝その記録をリモート（GitHub）へ“送る”

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
一目でなにをしたか分かる書き方をする

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
プルリクエストでマージだと、
リモート上でマージした内容を、ローカルのmasterブランチは知らず、
自動でリモートリポジトリとローカルリポジトリは同期されない事。


## コンフリクトを起こしてしまった場合、どう対処すべきですか？
現場の人間と話し合い、以下から選択。
・先にマージされた変更内容を取り込む
・後にマージしようとしている変更内容を取り込む
・どちらの変更内容も取り込む
